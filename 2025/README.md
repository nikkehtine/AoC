# Advent of Code 2025

https://adventofcode.com/2025

## Structure
- `example.txt` - Provided example for testing
- `input.txt` - Actual input for the challenge
- `main.*` - The program
- `Makefile` - Responsible for building and running the program
- `README.md` - Description
- `bin` - Compiled binaries
- `output.txt` - Generated answer

## How to run

`cd` into the directory and then run:

```sh
make run # Run the program
```

```sh
make test # Run the example
```

```sh
make clean # Clean up generated files
```
